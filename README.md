# 🧠 NeuroLab — Neural Network Toolbox

A **professional, dark-themed** Streamlit dashboard for exploring neural networks visually and interactively.

## Modules

| Module | Description |
|--------|-------------|
| ⚡ Perceptron Lab | Train single-neuron on AND/OR/XOR gates with animated decision boundary |
| ➡️ Forward Propagation | Step through MLP layer-by-layer, inspect activations & weight heatmaps |
| ↩️ Backpropagation | Visualize gradients, loss landscapes, optimizer comparison |
| 🎭 Sentiment RNN | Character-level LSTM with gate activation timeline |
| 📊 Architecture Viz | Design any topology and render live node-edge diagram |

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Stack
- **Streamlit** — dashboard framework
- **NumPy** — all ML implemented from scratch
- **Plotly** — interactive visualizations
- **Orbitron + Space Mono** — custom typography (Google Fonts)
